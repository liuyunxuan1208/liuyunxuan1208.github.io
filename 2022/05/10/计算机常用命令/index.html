计算机预科

1.打开CMD的方式

  1. 开始 + 系统 + 命令提示符。
  2. Win + R 输入 CMD 打开控制台 (推荐使用)。
  3. 在任意文件夹下，按住Shift键 + 鼠标右键打开命令行窗口。
  4. 在资源管理器的地址栏前加上 CMD 路径。

2.管理员身份运行方式

  • 选择以管理员方式运行。

3.常见的Dos命令

  1. 盘符切换

  2. 查看当前盘符目录下的全部目录：dir

  3. 切换目录：cd change directory

  4. 返回上一级：cd ..

  5. 清理屏幕： cls

  6. 退出终端：exit

  7. 查看电脑IP信息：ipconfig

  8. 打开计算机：calc

  9. 打开画图：mspaint

  10. 打开记事本：notepad

  11. ping命令：ping 网址

  13. 删除文件
